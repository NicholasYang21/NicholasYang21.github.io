什么是CS:IP

CS: Code Segment register 代码段寄存器

IP: Instruction Pointer register 指令指针寄存器

在x86 CPU中，地址加法器通过这两个寄存器的值获得指令的地址。

$CS \times 10H + IP = $指令地址

因为MBR只有$512Bytes$，所以需要跳出这片空间。

在x86中：

mov可以改变CS寄存器，但不可以改变IP寄存器。
